KineticaDB Operator Chart

This is the home of the KineticaDB Operator Helm Chart. The chart is hosted on GitHub Pages and can be added to your local helm repo with the following command:

helm repo add kinetica-operators https://kineticadb.github.io/charts

OnPrem Installation

Kind

To install the KineticaDB Operator on a Kind cluster, use the following commands:

kubectl create namespace kinetica-system

helm install --namespace kinetica-system kinetica-operators kinetica-operators/kinetica-operators --set provider=kind

Azure AKS

To install the KineticaDB Operator on an Azure AKS cluster, use the following commands:

kubectl create namespace kinetica-system

helm install --namespace kinetica-system kinetica-operators kinetica-operators/kinetica-operators --set provider=aks
